## 1. 入口与两种运行形态（关键:看是不是被 MODELING/STRATEGY 调用）
- 用户可能**直接甩一个 join 好的 csv 进特征分析**(跳过 JOIN)→ 开头跑 `infer_schema`,核对**目标列**与所选指标所需列是否齐;缺则提示。目标列在此确认一次(若经 JOIN 来则沿用 JOIN C1,不重复问)。
- **不强制 train/test/oot 切分**(切分只在 MODELING 必需;特征分析也供策略开发用,策略不需切分)。
- **两种运行形态(按入口区分,务必分清):**
  - **A·独立特征分析任务**(直接点欢迎页"特征分析"):用户**只要分析 + 报告,不做筛选**。按创建时所选指标算完 → **产出 Excel 特征分析报告** → 右侧任务栏**下载**(样式同模型验证的 Excel 下载)。**不弹筛选宽表,无勾选/滑杆/强选等操作。**
  - **B·被 MODELING/STRATEGY 调用**(那些流程走到"特征分析+筛选"步):**才**进入 §5 的**宽表筛选门**(滑杆/勾选/硬剔标色/强选确认/agent 预筛列),产出**选中特征集**交下游。
  - 共用 §2–§4 的指标计算;只有"是否进筛选门 + 产物(Excel 报告 vs 选中特征集)"不同。

## 2. 指标清单（创建任务弹窗勾选,选谁算谁）
| 指标 | 依赖 | 默认 |
|---|---|---|
| IV / KS / AUC | 目标列 | ✓ |
| coverage(覆盖率) | — | ✓ |
| PSI·分月-首月往后比 | 时间/月份列 | 可选 |
| PSI·分月-尾月往前比 | 时间/月份列 | 可选 |
| PSI·分月-每月比上月 | 时间/月份列 | 可选 |
| PSI·train/test/oot | 切分 | 可选 |
| 共线(collinear_pairs / VIF)| — | 可选 |
| 头部 lift5/10% · 尾部 lift5/10%(按**风险方向**定头尾)| 目标列 | 可选 |
| 特征重要性 | 无(7:3 随机)| 可选(选了才跑 LGB)|
| 含义正负向一致性 | **agent 模式 + 数据字典** | 可选(仅 agent + 有含义)|
- **全部指标皆勾选项,无强制**(含每种 PSI 视角);给常用默认勾选,可自由增减。**选什么算什么**。
- **不选某指标就完全不算**——例:不选特征重要性 → 不跑 LGB;不选任何 PSI → 完全不算 PSI。
- agent **按用户所选指标**做预筛(只用算了的指标)。

## 3. 计算子步骤（仅所选指标,连续跑不停）
- **基础指标**:`compute_feature_metrics` → IV/KS/AUC/coverage(逐特征,批量,内存可控)。
- **PSI(选了才算,每种视角独立勾选)**:分月-首月往后比 / 分月-尾月往前比 / 分月-每月比上月 / train-test-oot;输出每特征对应 PSI 序列 + 最大 PSI。无强制。
- **共线**:`correlation_analysis` → collinear_pairs(|corr|>阈值)+ VIF。
- **头部/尾部 lift(扩 `feature_lift`)——按风险方向定头尾**:
  - 先定特征与 y 的**风险方向**(y=1 为坏;算特征↔y 相关符号)。
  - 按"**越坏越靠头**"给特征定向:正相关→大值端为头;负相关→小值端为头(**头≠特征大值,而是风险高那端**)。
  - **头部** = 风险最高的 5%/10% 段、**尾部** = 风险最低的 5%/10% 段;lift = 段内坏率 / 整体坏率。
  - 头部 lift 高 = 能把坏客户浓缩在头部(强);尾部 lift 低 = 尾部很干净。
- **特征重要性(选了才跑)**:
  - 用**全部非硬剔特征**跑一个 **默认参数 LGB**(不调参)。
  - **内部 7:3 随机切 train/test**(固定 seed 可复现),**不按时间、不需时间列、无 OOT**。
  - 输出每个入模特征的**重要性**列,供筛选参考(**不**作硬筛依据)。
  - 注:此切分是该指标**内部专用**,与 MODELING 的用户确认切分无关。
- **含义正负向一致性(仅 agent + 字典)**:
  - 把特征含义(数据字典)喂 LLM → LLM 判该特征对 Y 应为 **正向 / 负向 / U 型 / 不确定**。
  - 算特征与 Y 的**实际方向**(单调:相关符号;判 U 型则用非线性/分箱坏率走势)。
  - 输出"是否一致"标签:不一致(如 学历应正相关收入、数据却为负)→ 标"可能有问题、不推荐"。

## 4. 筛选确认门 = 宽表弹窗（核心交互）
对话流里可点开的**宽表**:
- **行** = 每个特征;**列** = 所选各指标值(IV/KS/AUC/coverage/分月PSI/共线/4 个 lift/重要性/一致性…)+ **末列勾选框**;(agent 模式)多一列 **"agent 预筛建议"**(可点看原因)。
- **旁边滑杆**:IV / PSI / coverage 等阈值滑杆 → 滑动即**自动勾选**达标特征;用户也可**手动勾/取消**。
- **硬剔**(泄漏/常量/过稀)的行**标色 + 原因**;用户**硬选**(强制加回)→ 弹窗二次确认。
- **最终选择以勾选框为准**(agent 预筛只是参考列,不替你决定)。
- 一次性摊全表,看完一次确认 → 选定特征集。

## 5. 手动 vs agent
| | 手动模式 | agent 模式 |
|---|---|---|
| 指标 | 创建时勾选 | 同;LLM 也可建议补算 |
| 预筛列 | 无(纯阈值滑杆+手勾)| 多"agent 预筛建议"列 + 原因 |
| 含义一致性 | 不可用(需 LLM)| 可用(有字典时)|
| 调整 | 滑杆/勾选 | 滑杆/勾选 + 自由文本("把 PSI 阈值放到 0.1 再去掉 age")|
- 共用宽表 + 勾选框真值 + "重跑本步换参数"原语。

## 6. 输出 / 交接
- 选定特征集 + 全分析宽表(可下载)→ 交 **MODELING**(作筛后特征)或 **STRATEGY**。
- 分析宽表落 artifact,右栏可下载/预览。
